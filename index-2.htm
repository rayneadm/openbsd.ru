Файлы

/etc

DIR_COLORS Файл настроек раскраски для gnuls.
csh.cshrc, csh.login Startup файлы для c-shells (csh, tcsh).
pf.conf Пример настройки PF для простого маршрутизатора.
pf-dual.conf Пример настройки PF для двух каналов с симметричной маршрутизацией.
pf-home.conf Пример настройки PF для маршрутизатора в домашней сети с использованием очередей.
pf47-home.conf Пример настройки PF для маршрутизатора в домашней сети с использованием очередей (для OpenBSD 4.7).
pf-tunnel.conf Пример настройки PF для маршрутизатора с IPSec тунелями.
profile, zshenv Startup файлы для bourne shells (sh, ksh, bash, zsh).
 

/etc/isakmpd

A-isakmpd.conf, B-isakmpd.conf,
isakmpd.policy, README		 Пример настройки IPSec с использованием ISAKMP (IKE).
 

/etc/mail

Makefile, sample.mc, spam-domains, access Makefile для компиляции содержимого /etc/mail и пример конфигурации sendmail.
spamd.bypass Список сетей, которые нужно пропускать напрямую без проверки через spamd greylisting.
 

/etc/X11

xorg-vmware.conf Пример xorg.conf для настройки Xorg под VMWare.
 

/etc/ssh

ssh_known_hosts Пример ssh_known_hosts. Содержит ключи anoncvs серверов.
 

Файлы домашнего каталога

config Пример настройки .ssh/config.
cvsrc Пример настройки .cvsrc.
dovecot.sieve Пример файла правил для dovecot-sieve.
gtkrc Пример настройки .gtkrc для включения русского в gtk 1.x.x программах.
inputrc Пример настройки .inputrc для включения русского в readline программах.
kermrc Пример настройки .kermrc для связи kermit с Kermit-11 по telnet.
twmrc Пример настройки TWM с русскими шрифтами.
Xdefaults Пример .Xdefaults (русификация netscape, настройка и руссификация xterm, xcalc).
Xmodmap Пример использования мультимедия кнопок на клавиатуре.
 

Разное

gqmpeg.tar.gz GQmpeg Default Skin с поддержкой русских букв. (посмотреть).
RUSSIAN_WIN Файл трансляции koi8-r <> windows-1251 для irssi, ircII, BitchX, xchat и т.д.
site.tar.gz, README Набор для облегчения жизни тем, кто руками собирает порты и исходники (подробности).
userChrome.css Некоторые полезные настройки для Firefox.
VMWARE Пример конфига ядра i386 для работы под VMWare.
xmms.tar.gz XMMS-like Skin для GQmpeg с поддержкой русских букв. (посмотреть).
